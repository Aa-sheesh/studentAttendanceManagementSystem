# Setup
1. Open XAMPP and start MYSQL and Apache Server.<br/>
2. Upload the database file to MYSQL Admin. <br/>
3. Put the project file in C:/xampp/htdocs. <br/>
4. Run localhost/"projectname" on web browser. <br/>
